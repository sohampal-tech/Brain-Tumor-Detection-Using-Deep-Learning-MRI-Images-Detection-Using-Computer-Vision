📌 Project Overview

Brain Tumor Detection is a Deep Learning and Computer Vision based project that automatically detects brain tumors from MRI scan images. The system uses Convolutional Neural Networks (CNN) to classify MRI images into tumor and non-tumor categories with high accuracy.

This project helps doctors and medical professionals in early diagnosis and faster treatment planning by reducing manual effort and improving detection efficiency.

MRI Image Processing
Brain Tumor Classification
Deep Learning Model (CNN)
Computer Vision Techniques
Image Preprocessing
